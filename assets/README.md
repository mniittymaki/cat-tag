# Assets / Demos

This folder is for screenshots, GIFs, and demo videos used in the main README.

## Current / Planned Demos

- `transcription-demo.gif` — Shows the full AI transcription → translation → burning workflow
- `ai-tags-demo.gif` — Shows using "Ask AI for tags" from the grid or inspector
- `import-ai-demo.gif` — Shows AI auto-tagging during import with live progress

## Guidelines for new demos

- Keep GIFs under ~10MB when possible
- Show the full relevant UI (not just a cropped window)
- Use a clean test catalog when recording
- Prefer real footage over lorem ipsum when possible

When adding a new demo, update the main README.md to point to it.
